# OpenSpec 指南

这些说明用于本项目中的 AI 助手。

当需求满足以下任一条件时，必须先打开 `@/openspec/AGENTS.md`：
- 提到规划或提案（例如 proposal、spec、change、plan 等关键词）
- 引入新能力、破坏性变更、架构调整，或较大的性能/安全工作
- 需求含糊，需要先以权威规范为准再开始编码

使用 `@/openspec/AGENTS.md` 了解：
- 如何创建与应用变更提案
- 规范格式与约定
- 项目结构与指导

请保留此托管区块，以便 `openspec update` 能刷新指令文件。
