# 设计系统 — PineSnap

## 产品定位
- **产品描述：** AI 驱动的学习应用，将用户收藏的内容转化为交互式苏格拉底学习会话
- **目标用户：** 技术向学习者（开发者、工程师），习惯收藏内容但从不回看
- **行业/领域：** EdTech / AI 学习。竞品：Duolingo（游戏化）、Brilliant（交互优先）、Gemini Guided Learning（聊天式）、OpenMAIC（多智能体课堂）
- **产品类型：** Web 应用（Dashboard + 学习画布）
- **核心理念：** 推送而非拉取。用户不是来学习的，而是发现学习已经为他们准备好了。就像松鼠找到自己埋下的橡果。

## 美学方向
- **风格：** Organic/Natural（有机/自然）
- **装饰层级：** 有意为之 — 背景使用微妙的纸张纹理，整体色调温暖。不使用装饰性元素。
- **情绪：** 森林小屋里的学习空间。温暖、专注、有学术感但又平易近人。像发现一本恰好是你需要的书。
- **差异化：** 既不是游戏化（Duolingo）也不是严肃学术（Brilliant）。第三条路线："发现的惊喜"。
- **品牌隐喻：** 松鼠与橡果 — 囤积书签（收集）、AI 消化内容（敲开橡果）、学习会话（汲取养分）。如果后续改名为 ACORNA，视觉体系已就绪。

## 字体

- **展示/标题：** Fraunces（可变衬线体，支持光学尺寸） — 学术感但温暖，赋予产品"书籍感"，区别于典型的无衬线体应用
- **正文：** Plus Jakarta Sans — 清晰、可读性好，不像 Inter 那样泛滥
- **UI/标签：** Plus Jakarta Sans（与正文相同，标签使用 weight 600）
- **数据/表格：** JetBrains Mono（支持 tabular-nums）
- **代码：** JetBrains Mono
- **加载方式：** Google Fonts CDN
- **字号规范：**
  - 展示级：48px / weight 600 / line-height 1.15
  - 标题级：32px / weight 500 / line-height 1.25
  - 小标题：20px / weight 600 / line-height 1.35
  - 正文：16px / weight 400 / line-height 1.65
  - 小正文：14px / weight 400 / line-height 1.55
  - 说明/标签：11-12px / weight 600 / letter-spacing 0.12em / 大写
  - 代码：14px / weight 400 / line-height 1.6

## 颜色

### 策略：克制 + 签名强调色

森林绿是主交互色（按钮、聚焦、选中态）。橡果琥珀色仅用于两个情感时刻：**发现**（AI 找到了知识关联）和 **完成**（学习会话结束）。这种稀缺性让琥珀色每次出现都有意义。

### 橡果色使用规则
- **使用场景：** 环境式学习通知卡片、知识卡片（完成态）、"发现关联"标签、完成时的概念标签
- **禁止使用场景：** 常规按钮、表单聚焦态、选中的答题选项、导航、链接
- **原因：** 琥珀色应该感觉像一个特别的时刻，而不是一个普通的 UI 颜色。如果到处都是，就失去了意义。

## 间距
- **基础单位：** 8px
- **密度：** 舒适 — 学习需要呼吸空间
- **规范：** 2xs(2px) xs(4px) sm(8px) md(16px) lg(24px) xl(32px) 2xl(48px) 3xl(64px)

## 布局
- **策略：** 全屏学习画布 + 抽屉式聊天
- **学习画布：** 默认全屏，内容居中于 max-w-2xl（640px）。体验驱动，参考 Brilliant。
- **聊天抽屉：** 默认关闭，右侧滑出。承载自由对话、AI 思考过程。像游戏里的聊天日志。
- **Dashboard：** 内容居中于 max-w-3xl（768px）
- **最大内容宽度：** 外层容器 960px
- **底部操作栏：** 答题反馈 + 操作按钮（"Why?" / "Continue"），参考 Brilliant
- **圆角：** sm(4px) md(8px) lg(12px) xl(16px) full(9999px)
- **响应式：** 桌面端优先（1440px）。移动端为 Phase 2。组件使用 max-w + 百分比，不用固定宽度。

## 动效
- **策略：** 有意为之 — 平静且有目的性，不弹跳、不游戏化
- **缓动函数：** 进入(ease-out) 退出(ease-in) 移动(ease-in-out)
- **时长：** 微(50-100ms) 短(150-250ms) 中(250-400ms) 长(400-700ms)
- **关键动画：**
  - 内容阶段过渡：旧内容淡出（200ms），新内容淡入（300ms）
  - AI 思考指示器：三个森林绿圆点脉动（无限循环，1.5s 周期）
  - 答题选项选中：边框颜色过渡（150ms ease-out）
  - 面包屑概念完成：微妙的缩放 + 颜色变化（200ms）

## 背景纹理
- 浅色模式：在 `--cream` 背景上叠加 1.5% 透明度的 SVG 分形噪声。赋予"纸张感"但不厚重。
- 深色模式：不使用纹理。干净的深色表面。
