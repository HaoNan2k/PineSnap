# 连接 Bilibili（MVP）

## 这项连接能做什么？

连接后，你可以在 B 站视频页一键采集字幕并写入 PineSnap：

- 采集字幕：以页面可见内容为准（MVP）
- 写入 PineSnap：会存入素材库（Resource），供后续整理使用

## 如何连接（一次性启用）

1. 登录 PineSnap
2. 右上角头像菜单 → **连接 Bilibili**
3. 点击 **一键启用（安装浏览器连接器）**
4. 浏览器会提示安装/更新，确认即可

## 以后要怎么操作？

- **更换电脑/浏览器**：重新执行一次“连接 Bilibili”即可
- **想停用/怀疑泄露**：进入“管理/断开连接”，点击“断开连接”（断开后立即生效）
